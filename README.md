# SQL Analytics Lab

Repositório dedicado a estudos, exercícios e práticas de SQL voltadas para análise de dados, construção de relatórios e cenários analíticos utilizados em BI e Data Analytics.

Este projeto reúne consultas desenvolvidas durante minha jornada de aprendizado em SQL, com foco em lógica analítica, manipulação de dados e resolução de problemas aplicados ao contexto corporativo.

---

# 🎯 Objetivos

* Fortalecer conhecimentos em SQL aplicado à análise de dados
* Evoluir em construção de consultas analíticas
* Melhorar lógica de joins, agregações e filtros
* Praticar cenários comuns de BI e reporting
* Desenvolver consultas mais organizadas e performáticas

---

# 📚 Conteúdos praticados

## Consultas básicas

* SELECT
* WHERE
* ORDER BY
* DISTINCT

## Manipulação e análise

* JOINs
* GROUP BY
* HAVING
* CASE WHEN
* UNION ALL
* Subqueries

## Funções analíticas

* Window Functions
* RANK()
* ROW_NUMBER()
* SUM() OVER()
* COUNT() OVER()

## Tratamento de dados

* Tratamento de NULL
* Conversões de tipo
* Filtros por período
* Validação de duplicidades
